Linked List

Linked list is a linear data structure in which the elements are not stored at contiguous memory locations. Instead, each element points to the next element using a pointer.

Binary Tree

Binary trees are mainly used for searching and sorting as they provide a means to store data hierarchically. Some common operations that can be conducted on binary trees include insertion, deletion, and traversal.

Stacks

A stack is a linear data structure that follows the principle of Last In First Out (LIFO). This means the last element inserted inside the stack is removed first. You can think of the stack data structure as the pile of plates on top of another.

Queues

Queue is used when things don't have to be processed immediately, but have to be processed in First In First Out order like Breadth First Search. When a resource is shared among multiple consumers.

Blockchain

Blockchain is a system of recording information in a way that makes it difficult or impossible to change, hack, or cheat the system. A blockchain is essentially a digital ledger of transactions that is duplicated and distributed across the entire network of computer systems on the blockchain.

Github branches

Branches allow you to develop features, fix bugs, or safely experiment with new ideas in a contained area of your repository. Typically, you might create a new branch from the default branch of your repository.

Web Tab Navigation

A Linked list can be used to store a tab address and when user clicks on a link it will create a new node and append it to the head node.

Music Player

Whenever we are playing music in a sequence, a node is a current music that we are playing and next song is what a node.next connected to.

Matching text expressions

Regular Expression(sometimes also called the Rational Expression ) is a sequence or set of characters that define a search pattern, mainly for the use in pattern matching with strings or string matching

Backtracing

Each "stack frame" is the section of stack used by a particular function, so there's one frame per function. Since the current function is main() , there is only one frame to show.

Memory Management

Stack memory is a memory usage mechanism that allows the system memory to be used as temporary data storage that behaves as a first-in-last-out buffer.

Scheduling

As processes enter the system, they are put into a job queue, which consists of all processes in the system. The processes that are residing in main memory and are ready and waiting to execute are kept on a list called the ready queue.

Networking

Networks of queues are systems which contain an arbitrary, but finite, m number of queues. Customers arrive at the service centre, wait in the queue if necessary, receive service from the server, and depart.

File System

Queue manager data is stored in the file system. A queue manager makes use of file system locking to prevent multiple instances of a multi-instance queue manager being active at the same time.

Searching

Searching in a tree is alot faster then searching in a list. It takes O(N) time if we search through the list, but it takes O(logN) time if we search through the tree.

Sorting

Sorting an array is O(N^2) time complexity. Sorting a tree is O(NlogN) time complexity which is faster and more effictive way to do it.
